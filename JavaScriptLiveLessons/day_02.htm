Common Input Attributes

Attribute Description
type Specifies the input type.
name Name used when submitting form data.
id Unique identifier.
value Default value.
placeholder Hint text inside input.
required Makes field mandatory.
readonly User cannot modify value.
disabled Disables input.
maxlength Maximum number of characters.
minlength Minimum number of characters.
min Minimum allowed value.
max Maximum allowed value.
step Increment interval.
size Visible width of input.
pattern Regular expression validation.
checked Default selected checkbox/radio.
multiple Allows selecting multiple values/files.
accept Allowed file types.
autocomplete Enables/disables suggestions.
autofocus Automatically focuses input.
list Connects input with datalist.
form Associates input with a form.
src Image source (type="image").
alt Alternate text for image input.
dirname Stores text direction.
